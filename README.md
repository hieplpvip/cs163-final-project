## Build Instruction

This project uses C++20. You need CMake to build it.

### On Linux

```shell
cmake .
make
```

To run it:

```shell
./mse
```

### On Windows

Open this directory with Visual Studio. It will automatically generate CMake cache.

On the menu bar, choose **Build** -> **Build All**.
